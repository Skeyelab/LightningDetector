📁 Manual Firmware Upload

If automatic release detection fails, you can manually upload firmware files:

Note: You can download firmware files from:

  • GitHub Releases (if available)
  • Build firmware locally using PlatformIO: pio run -e sender or pio run -e receiver
  • Check the .pio/build/ directory for firmware.bin files

🔧 Flashing Instructions

  1. Connect your Heltec V3 board via USB cable
  2. Put the board into download mode by holding the BOOT button while pressing RESET
  3. Release the RESET button, then release the BOOT button
  4. Select your firmware type above
  5. Click the "Flash Firmware" button below
  6. Wait for the flashing process to complete
  7. Reset the board after flashing is finished
